## Bilateral Trade Measure

The trade network is constructed using bilateral exports obtained from the IMF Direction of Trade Statistics (DOTS).

Each directed edge represents exports from one G7 country (exporter) to another G7 country (importer). The network therefore illustrates the magnitude and direction of trade flows between G7 members.
